FAQ

Why did you make this theme?

The initial proposal for this theme was made to fix Shaxzy's theme, which you can find here (deprecated). My intention was to update the elements to the new recode. But when I saw his CSS I realized that it was going to be better if I make a CSS from scratch. I thought it may look better if the style tries to mimic the look of skeet, so while making the theme I was watching skeet screenshots instead of the old theme. This idea came to my mind because Discord also has a sidebar and I wanted to see Discord being skeet.
Still, I rescued the window border and the title gradient because I didn't see a better way to do it.

Is this your first Discord theme?

Yeah. But it's not the first page I make a theme for. I've been learning CSS for a long time and I've done more themes for, among other reasons, practice. You can find some of my themes here.

Why this theme looks so bad?

Before using it, make sure you go to the Appearance settings on the Discord client and set the "Chat font scaling" scroll to 14px. This happens because I worked with it and at the moment I'm too lazy to fix it right now.
Also, note that I've only themed the elements the people use the most. There might be things like the "Friends" tab that are untouched.
